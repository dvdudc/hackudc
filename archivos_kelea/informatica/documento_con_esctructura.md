# Arquitectura del proyecto del hackaton

## Objetivo
Crear un sistema de documentación personal con separación clara entre captura y procesamiento.

## Decisiones técnicas
- Backend en FastAPI
- Almacenamiento en Markdown
- Búsqueda basada en TF-IDF
- IA opcional para resumen
