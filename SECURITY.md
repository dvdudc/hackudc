# Security Policy

## Supported Versions

Currently, as we are in active early development, only the latest `main` branch and the latest minor tag (e.g. `v0.3.x`) receive security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.3.x  | :white_check_mark: |
| v0.2.x  | :x:                |
| < 0.2   | :x:                |

## Reporting a Vulnerability

We take the security of Black Vault seriously. If you discover a security vulnerability in this project, please report it directly to the maintainers instead of opening a public issue.

Please include the following information in your report:
- A detailed description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact and an assessment of the severity.

We will try to acknowledge your report within 48 hours and we will work with you to understand and resolve the issue as quickly as possible. We kindly ask that you do not publicly disclose the vulnerability until an official patch has been released.
